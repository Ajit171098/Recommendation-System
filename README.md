# Recommendation-System

Workflow (order in which we update files)
1. config.yaml
2. entity
3. config/configuration.py
4. componenets

6. main.py // just to test the pipeline

# How to run

### Step 01 - create a conda environment after opening the repository

```bash```

conda create -name recommendationsystem python=3.7.10 -y

```bash```
conda activate recommendationsystem

### Step 02- install the requirements

```ignore if you have installed anaconda```

pip install -r requirements.txt

### Run app.py file

``` app.py```
